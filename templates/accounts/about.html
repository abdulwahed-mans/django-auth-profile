About AuthProfile

A modern Django starter for authentication and user profiles.

What is this?

AuthProfile is a ready-to-use Django project that provides user registration, login/logout, editable user profiles, and a full REST API. It's designed as a clean starting point for any project that needs user management.

The project follows Django best practices with a separate config directory, organized apps folder, and clean separation of concerns between views, serializers, and templates.

Tech Stack
Backend Django 5.2
API DRF 3.16
Database SQLite
Frontend Bootstrap 5
Fonts Inter, Jakarta Sans
Auth Session + Token
Features
User registration
Login & logout
Auto-created profiles
Profile editing
REST API with CRUD
Token authentication
Browsable API
