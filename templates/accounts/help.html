Help Center

Answers to common questions about using AuthProfile.

Getting Started
How do I create an account?

Click Register in the top navigation bar. Fill in your username, name, email, and password, then click "Create account". You'll be logged in automatically.

How do I log in?

Click Login in the navigation bar and enter your username and password. You'll be redirected to your dashboard.

How do I edit my profile?

Go to your Dashboard and click "Edit Profile", or click Profile in the navigation bar. You can update your name, email, bio, location, and phone number.

REST API
API Endpoints
GET /api/profiles/ List all profiles
GET /api/profiles/{id}/ Get a single profile
PUT /api/profiles/{id}/ Update a profile
DELETE /api/profiles/{id}/ Delete a profile
GET /api/users/ List all users (read-only)
Authentication

The API supports both session authentication (when logged in via browser) and token authentication. To use token auth, include the header:

Authorization: Token your-token-here
Browsable API

Visit /api/ in your browser to explore the API interactively using DRF's built-in browsable interface.
